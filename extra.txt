# 1. Inisialisasi folder jadi repo Git
git init

# 2. Masukkan file index.html ke 'staging'
git add index.html

# 3. Kunci perubahannya
git commit -m "initial commit: map logic and firebase"

# 4. Arahkan ke branch main
git branch -M main

# 5. Hubungkan ke GitHub (Ganti [URL] dengan link dari GitHub tadi)
git remote add origin https://github.com/username-kamu/peta-desa-rt-rw.git

# 6. Upload!
git push -u origin main

//
Paling penting. ada history di atas kanan biar nanti bisa baca apa yang di ganti
//

Edit File Langsung:

    Klik file index.html.

    Klik ikon Pensil (Edit this file) di pojok kanan atas.

Update Config/Data:

    Cari bagian firebaseConfig, ganti pakai punya kalian.

    Kalau mau ganti teks pop-up, cari bagian kodenya di bawah.

Simpan:

    Klik tombol hijau "Commit changes...".

    Klik "Commit changes" lagi.

Cek Hasil:

    Tunggu 1 menit, buka link GitHub Pages-nya. Webnya bakal otomatis update sendiri.
